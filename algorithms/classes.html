DataConverter¶

class dataConverter.DataConverter(dataImport)¶
Author of class

Fabian Waldvogt

This class constructs (or loads) a data structure (matrix that holds all distances from one node to another) from the list from DataImport.getData() with which all routing algorithms work.

Member

data: DataImport Instance of DataImport which provides the initial node lists

CONST_ALLTRASHNODES: list A list of all nodes with the attribute trash set to true

CONST_ALLNODES: list A list of all nodes

self.CONST_ROUTINGDATA: dict A matrix consisting of two dictonaries with nodes as keys that hold the distances from every trash node to every other one

self.CONST_PREVIOUSNODESDATA: dict A matrix consisting of two dictonaries with nodes as keys that hold the previous node(node through which other node was discovered) to every other node with this a full route (all nodes) can be constructed out of a route of only trash nodes

__init__(dataImport)¶

This method initializes the class DataConverter and either constructs a new datastructue or loads an old one(if there were no updates)

Parameters

dataImport: DataImport Instance of DataImport which provides the initial node lists

singleSourceShortestPathsBinaryHeap(source)¶

Executes dijkstra’s single soure shortest path with the source as start node with a heap dictonary, that is used as a priority queue, as the underlying data strucutre

Parameters

source: Node The starting node from which dijkstra’s algorithm will begin

Returns

distances dict , previousNode dict , source Node

transformData()¶

Creates the new data structure: Executes dijkstra’s single soure shortest path for all trash nodes in CONST_ALLTRASHNODES and saves the returned dictonaries in the dictonaries self.CONST_ROUTINGDATA, self.CONST_PREVIOUSNODESDATA with the current node as key then saves the data to pickle files and returns the new data structures

Returns

distancesDict <-> self.CONST_ROUTINGDATA dict, previousNodesDict <-> self.CONST_PREVIOUSNODESDATA dict
