API Reference¶

The following section outlines the API of OSM DATA

Note

This module uses the Python logging module to log diagnostic and errors in an output independent way. If the logging module is disabled or the loginlevel is set to high then these logs will not be output anywhere.

Version Related Info¶

The current Version only generates random Data. There will be changes and missing Features in this Release

OSM Data¶

class osm_data.OSM(demo_amount: int = 100, skip_update: bool = False, loggin_level: int = 20)¶

Represents the OSM Data in an easy to access format. The data gets currently randomly generated instead of real data being used.

Parameters

demo_amount Optional[int]

The amount of streets and Nodes you want to generated

skip_update Optional[bool]

Defines if the update check should be skipped

loggin_level Optional[int]

Defines which logging level should be used

Attributes

nodes List[Node]

Returns all Nodes of this Object

ways List[Way]

Returns all Ways of this Object

Node¶

class osm_data.Node(id: int, nodetype: osm_data.Enum.NodeType, ways: typing.List[osm_data.Way.Way], trash: bool, parent)¶

Represents a Node in the Dataset

Parameters

id: int

The unique id of the Node

nodetype: NodeType

Defines the Type of the Node

ways List[Way]

All ways that are connected to this Node

trash bool

Defines if there is a trash container on this Node

parent OSM

The main object that controls this Node

Attributes

connected_ways List[Way]

Returns all ways that are connected to this Node”

connected_nodes List[Node]

Returns all Nodes that are connected with this Node

Warning

This might be removed in the future. Please give Feedback on this Option

trash bool

Returns if the current location contains a trash container

Way¶

class osm_data.Way(id: int, name: str, distance: int, speed: int, waytype: osm_data.Enum.WayType, parent: osm_data.OSM.OSM)¶

Represents a Way in the Dataset

Parameters

id: int

The unique id of the Way

name str

The Streetname

distance: int

The length of the Street

speed: int

The speed limit of the Street

waytype WayType

Defines if the current Street is a One-Way Street

parent OSM

The main object that controls this Node

Attributes

weight int

Weight of the Way

connected_nodes List[Node]

Returns all Nodes that are connected with the current Way

waytype WayType

Returns if the current Way is a One Way Street

Warning

Only returns normal Streettypes for now
